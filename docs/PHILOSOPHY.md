# Philosophy

Foundry is not a prediction engine.

It is a simulation framework for exploring how adaptive agents behave under uncertainty.

## What Foundry studies

```text
Event → Reaction → Decision → Outcome → Memory → Future Decision
```

## Design standard

Every feature should improve at least one of these:

- Explainability
- Reproducibility
- Inspectability
- Research value

Complexity is allowed only when it helps the simulation teach us something.
