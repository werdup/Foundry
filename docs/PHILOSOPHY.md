# Philosophy

Foundry is not a prediction engine.

It is a simulation framework for exploring how adaptive agents behave under uncertainty.

## Core belief

Every experiment inside Foundry should answer a question.

Not:

> Can we add a cool feature?

But:

> What does this help us understand?

## Example research questions

- Which strategies survive across many simulated worlds?
- How do memories change future decisions?
- How much of success is strategy versus luck?
- Do agents become more superstitious after near misses?
- Which behaviors are stable across many seeds?
- What beliefs emerge when agents adapt over time?

## Design standard

Every new feature should make the simulation:

- More explainable
- More reproducible
- More inspectable
- More interesting

Complexity is allowed only when it teaches us something.
